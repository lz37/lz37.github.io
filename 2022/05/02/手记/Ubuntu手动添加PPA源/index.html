Ubuntu手动添加PPA源（以apt-fast为例）

发表于2022-05-02|更新于2022-07-24|手记
|字数总计:355|阅读时长:1分钟|阅读量:

先说一下应用场景：

Ubuntu安装PPA源有时会很慢，等了半天也没个反馈，这时候就需要我们手动添加PPA源了。

参考：https://www.tok9.com/archives/480/

以安装apt-fast为例，

第一步添加PPA源的时候就出现了问题，这速度实在太慢了，纳闷我们就需要手动添加PPA源了。

首先访问PPA源官网: https://launchpad.net/

说一下PPA源的格式，PPA 的一般形式是： ppa:user/ppa-name

user：这个软件包的维护人 ppa-name： ppa软件名

利用这两个信息进行搜索，跳转到对应页面即可看到详细信息，对比确认无误，点开"Technical details about this PPA"。

终端输入"lsb_release -a"命令查看相应的版本，然后点击"Display sources.list entries for:"后面的下拉框选择对应的版本。

运行

sudo vim /etc/apt/sources.list

在sources.list文件末尾添加内容

deb https://ppa.launchpadcontent.net/apt-fast/stable/ubuntu focal main
deb-src https://ppa.launchpadcontent.net/apt-fast/stable/ubuntu focal main

注意版本

添加key，后面的一长串复制自Fingerprint的内容

sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys A2166B8DE8BDC3367D1901C11EE2FF37CA8DA16B

更新一下

sudo apt-get update && sudo apt-get upgrade

可以安装了

sudo apt-get -y install apt-fast
