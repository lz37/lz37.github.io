之前是用Gnome桌面插件Vitals来查看硬件、进程、网络信息的（进程没有），但是随着Gnome插件越装越多，渐渐暴露出了一个问题：

顶部栏越来越臃肿了（别看中间还有一个空，再加上Vitals基本就满了）

这时突然想到之前用Deepin的时候安装过Conky，于是就打算把监控信息用conky来展现，给顶部栏腾点地方。

安装过程如下

sudo apt install conky
