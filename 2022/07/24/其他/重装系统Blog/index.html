恢复

scoop 恢复

重装系统之后，可能会出现没有 wifi 的情况，这种时候使用手机给电脑共享 wifi，先让电脑安装完驱动

电脑可以连上网络之后，此时还不可以科学上网，scoop 的安装就无从谈起，不过可以打开手机端的 clash，打开allow lan选项，连接上电脑热点，电脑端的代理设置为手机的ip:clash端口，便可以科学上网了

首先恢复 scoop，这里要注意，为了恢复 scoop，我们需要先下载安装 git，也可以通过 scoop 安装，总之不要一上来就直接scoop import，否则报错之后还是得要按照我的步骤走。

安装好 git 之后，先行用 scoop 下载 clash 和 aria2

备份的配置文件归位好，确认没有问题后，便可以进行下一步操作

运行完毕后，便可以恢复 rainmeter 以及其他的软件配置了

WSA 上一般也不会（也不建议）安装很多安装程序，所以安装了什么用脑子或者笔记一下就行了，具体操作参考此链接

更加强大的 windows

淘宝上 35 块钱一个企业版的 key，可以将电脑升级为企业版

所以建议重装系统的时候选择 windows 专业工作站版，然后升级为企业版，这样在之后升级的时候下载的东西会少一些

升级完毕之后选择 windows 更新 > windows 预览体验计划，选择 beta 频道，体验最边缘的 windows

windows 预览版的一些功能：文件资源管理器的标签功能、WSLg（支持 gui 的 WSL，虽然我已经不用了^^）。

升级成功之后注意一些文件可能并没有同步到新的 windows，而仍然残留在windows.old文件夹，比如开始菜单里的scoop apps文件夹和shell:startup启动项，注意复制一下
