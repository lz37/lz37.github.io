社会主义革命理论{新民主主义社会改造的总路线和经验社会主义制度的确立社会主义革命理论\begin{cases} 新民主主义社会\\ 改造的总路线和经验\\ 社会主义制度的确立\\ \end{cases} 社会主义革命理论⎩⎨⎧​新民主主义社会改造的总路线和经验社会主义制度的确立​

3.1 新民主主义社会

3.1.1 新民主主义社会的特征和主要矛盾

  1. 社会形态特征
    • 非独立、过渡性
    • 属于社会主义体系
  2. 主要矛盾: 社会主义道路&资产阶级道路、工人阶级&资产阶级

3.1.2 新民主主义社会的经济成分

非社会主义→ 半社会主义/过度→ 社会主义
个体经济→ 合作社经济→ 集体经济、国营经济
私人资本主义经济→ 国家资本主义经济→

3.2 党在过渡时期的总路线

3.2.1 核心目标(两个转变)

  1. 实现从农业国向工业国转变
  2. 实现从新民主主义社会向社会主义社会的转变

3.2.2 两种工业化模式

  1. 英美资本主义模式：发展个体经济和私人资本主义
  2. 苏联社会主义模式：发展集体/国营经济

3.2.3 选择社会主义模式的原因

  1. 国内：国有经济主导且强大，资本主义很弱小
  2. 国际：资本主义国家的限制，社会主义国家的帮助
  3. 效果：成效快且可以摆脱成为附庸

3.2.4 总路线的提出：从1953年开始进行“一化三改”

  1. 一化是主体，三改是两翼
  2. 工业化和改造并举

3.3 社会主义改造的经验

3.3.1 农业改造的经验

  1. 积极引导农民走互助合作道路
  2. 自愿互利、典型师范和国家帮助原则
  3. 根据农村阶级和阶层状况制定政策
    • 依靠贫下中农推行互助合作
    • 限制并消灭富农
  4. 坚持稳步推进，循序渐进的方针
    • 互助社(集体生产)
    • 初级社(按劳分配的土地收入分红相结合，耕畜农具给予报酬)
    • 高级社(生产资料集体所有，按劳分配)

3.3.2 手工业改造的经验

  1. 方针：积极领导，稳步推进
  2. 经历由小到大，由低到高的三个步骤
    • 供销小组(包供包销)
    • 供销合作社：生产资料公有，合作社生产干预
    • 生产合作社：集体所有，统一经营，按劳分配

3.3.3 工商业改造的经验

  1. 方法：和平赎买(有偿、非支付、部分利)
    • 民资具有两面性(剥削+接受改造)
    • 党和民资长期统一战线
    • 民资对国营经济具有依赖性(大形势要求改造)
  2. 采取从低级到高级的国家资本主义的过渡形式
    • 初级形式国家资本主义：委托加工、统购包销
      四马分肥(税、金、福利、红利)
    • 高级形式的国家资本主义：公私合营
      • 个别企业公私合营(半社会主义性质)
      • 全行业公私合营(社会主义性质)
  3. 把资本主义工商业者改造为社会主义劳动者
    • 原则：“量才使用，适当照顾”
    • 改造阶级成分消灭资产阶级
      • 改造企业与改造人相结合
      • 改造资本家与消灭其所属资本相结合

3.3.4 经验总结

  1. 坚持社会主义工业化建设和社会主义改造并举
  2. 采取积极引导、逐步过渡的方式
  3. 用和平方法进行改造

3.3.5 历史反思

  1. 改造形式单一，管理配套不到位
  2. 要求过急，工作过粗，改变过快

3.4 社会主义制度的确立

时间：1956年；标志：三大改造完成

意义

  1. 为当代中国奠定了政治与经济基础
  2. 劳动人民成为国家主人，提高了人民的积极性、创造性
  3. 提供了和平改造的经验

11.3.1 为什么要依法治国

  1. 适应陌生人社会的发展需求
  2. 适应现代法治社会的发展趋势
  3. 更好地规范和约束公权力

1.《共同纲领》

  1. 1982年《中华人民共和国宪法》

  1. 中特法治道路核心要义

    • 贯彻中特法治理论
  2. 依法治国→\rightarrow→全面依法治国

11.2.3 全面依法治国的实践

11.2.3.1 全面的内涵

  1. 从法律体系到法治体系的全面转变
    • 法律体系(2011)→\rightarrow→法治体系(2014) (文本→\rightarrow→实践)
    • 法治体系：法律规范体系、法治实施体系、法治监督体系、法治保障体系
  2. 从执法到法律运行全过程的转变
    有法可依→\rightarrow→科学立法
    有法必依→\rightarrow→严格执法
    执法必严→\rightarrow→公正司法
    违法必究→\rightarrow→全民守法

11.2.3.2 全面依法治国的几个重要问题

  1. 坚持党对全面依法治国的领导
    • 伪命题：党大还是法大？（政治与法治）
    • 真命题：权大还是法大？
    • 结论：把权力关在制度的牢笼里
  2. 德治和法治的关系
    • 传统：德主刑辅
    • 当代：法主德辅（依法治国，以德治国）
  3. 为何要保证科学立法
    • 做好违宪审查工作
    • 案例：孙志刚案（2003）
      《城市流浪乞讨人员收容遣送办法》（1982）
      《城市生活无着的流浪乞讨人员救助管理办法》
  4. 为何要坚持严格执法
    • 按照法律要求严格执行到位
    • 严格按照法律程序执行
      案例：上海钓鱼执法（2009）
  5. 为何要坚持公正司法
    • 司法部门：法院+检察院+公安
      案例：聂树斌案
      1995年聂树斌因故意杀人罪被判死刑，2005年王书金称自己是杀人凶手，2016年改判聂无罪
    • 加大法治工作队伍建设
    • 保障法院司法独立

    • 符合近代政治实践
  2. 党的领导是中国特色社会主义最本质特征
    • 关系中国社会的性质、方向和命运
    • 是实现社会主义现代化和民族复兴的根本保证

    • 方法：自我革命（党内监督，显著标致）+党外监督（各党派、人民）

    1941年江纵队港九大队抗日武装军
  3. 自强不息的狮子山精神
    • 特点：依赖“自由市场”，政府干预较小
    • 结果：教育、医疗、金融投资明显增长，香港称为“亚洲四小龙”之一

（二）香港的归属问题

  1. 马岛战争胜利的撒切尔
  2. 1982年撒切尔与邓小平的谈判（铁娘子遇到钢铁公司），提出“一国两制”

  2. 香港金融业、服务业进入新阶段

  1. 贫富差距加大
  2. 产业金融化加剧
  3. 产业结构不合理
  4. 利益集团丛生，严重阻碍香港发展

（五）2019年香港问题的关键

  1. 去殖民化不彻底，没有掌握意识形态领导权（教育、媒体）
  2. 对司法权的失控（各国情报人员丛生）
  3. 香港本地人的内心失衡

  1. 通过香港国安法
  2. 调整香港选举制度：爱国者治港
  3. 最新香港行政长官：李家超

    • “另起炉灶”

  3. 外交思想
    • 和平共处五项原则
    • 三个世界划分
    • 一条线、一大片

  1. 外交思想
    • 和平和发展是时代的发展
    • 经济全球化是世界发展的趋势

    • 以公平争议为理念引领全球治理体系改革
